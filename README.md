# Biblioteca HTML Pública — GitHub Pages

Este pacote contém uma estrutura pronta para publicação no GitHub Pages.

## Arquivos principais

- `index.html`: página inicial com links para todas as páginas.
- `.nojekyll`: impede processamento Jekyll desnecessário.
- Demais arquivos `.html`: páginas estáticas preservadas com nomes seguros para URL.

## Publicação manual

1. Crie um repositório público no GitHub. Sugestão: `biblioteca-html-publica`.
2. Envie todo o conteúdo desta pasta para a raiz do repositório.
3. Vá em Settings > Pages.
4. Em Build and deployment, selecione Deploy from a branch.
5. Selecione branch `main` e pasta `/(root)`.
6. Salve.

URL esperada para repositório de projeto:

`https://elcidamb2025.github.io/biblioteca-html-publica/`

Se o repositório for `elcidamb2025.github.io`, a URL esperada será:

`https://elcidamb2025.github.io/`
